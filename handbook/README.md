# AI Assisted Development Platform ハンドブック

> プロジェクト No.0

## 概要

Development Platform は、個人の開発基盤として設計されたプロジェクトです。
再利用可能で AI アシスト機能を備えたソフトウェア開発環境を確立することを目的としています。

このハンドブックは、日本語ユーザー向けに以下の情報を提供します：

- プロジェクト概要
- AI 協業ルールと役割定義
- 開発ロードマップ
- アーキテクチャ設計ガイド
- 用語集

## AI 協業モデル

| AI         | 役割                                  | 強み                          |
| ---------- | ------------------------------------- | ----------------------------- |
| Claude     | 要件エンジニア / ドキュメント専門家   | 構造化、長文コンテキスト対応  |
| ChatGPT    | システムアーキテクト / 設計レビュア   | 広範な知識、コード生成        |
| Gemini     | リサーチアナリスト / 技術情報リサーチ | 最新情報、Google エコシステム |
| MS Copilot | コーディングアシスタント              | VSCode 統合、コード補完       |

> 詳細は [docs/operations/ai-roles.md](../docs/operations/ai-roles.md) を参照

### 協業原則

AI エージェントは専門的なアドバイザーとして機能します。
最終決定は、文書化された基準に基づいてプロジェクトオーナーが行います。
